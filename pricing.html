Frequently Asked Questions

Do I need to know how to code?

No. Describe your game to any AI assistant (ChatGPT, Claude, Gemini) in plain English. The AI generates the .raku file, and the runtime plays it instantly. You can learn game dev concepts by reading the generated code.

What AI platforms work with RakuAI™?

Any AI that can generate JSON works. We've tested with ChatGPT (GPT-4, GPT-4o), Claude (Opus, Sonnet), Google Gemini (Pro, Ultra), and GitHub Copilot. The SDK includes prompt templates optimized for each platform.

How big is the runtime?

Lightweight and fast. Works on Windows 10/11. XR features require a compatible headset.

Can I sell games I create?

Yes. You own the games you create. Free tier games can be shared freely. Pro tier includes standalone export for commercial distribution on Steam, itch.io, or any platform.

How does AI compliance work?

Every .raku game includes a compliance section for platform requirements. It declares which AI features are used so players and storefronts are informed. This is built into every template automatically.
