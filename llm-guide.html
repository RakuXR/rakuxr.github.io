With the SDK, you get access to our validated schema, prompt templates, and auto-fix tools that ensure AI-generated game files work perfectly every time. Our workflow is optimized for ChatGPT, Claude, Gemini, and other leading AI assistants.

Simple Three-Step Workflow

1. Prompt → Describe Your Game

Use any AI assistant (ChatGPT, Claude, Gemini, Copilot) with our SDK-provided prompt templates. Describe your game concept in natural language. The AI generates a valid .raku game file.

2. Validate → Ensure Quality

The SDK includes validation tools that automatically check and fix common issues in AI-generated files. One command ensures your game is ready to play.

3. Play → Instant Gameplay

Open the validated .raku file and play instantly. The Raku runtime handles everything from rendering to AI-driven adaptive gameplay.
