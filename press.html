Memberships

NVIDIA Inception

Member • 2026

RakuAI is a member of NVIDIA Inception. We are using the program's NGC catalog, partner cloud credits, and Deep Learning Institute training to bring NVIDIA-accelerated 3D Gaussian splatting (the Brush trainer on A10G / L40S) to Raku Capture - so a phone scan of any room turns into an explorable splat in minutes instead of an hour.

NVIDIA Inception is a free program for AI startups; RakuAI's membership does not imply that NVIDIA endorses RakuAI's products. Learn more at nvidia.com/startups.

AWS Activate

AWS Activate participant • 2026

RakuAI is an AWS Activate participant. Raku Capture and the spatial AI backend run on AWS infrastructure — GPU training (G5 instances), object storage, and compute for splat generation are powered by AWS. Activate gives early-stage startups credits and technical support to build on the AWS cloud.

AWS Activate is a program for startups; participation does not imply that Amazon Web Services endorses RakuAI’s products. Learn more at aws.amazon.com/activate.
