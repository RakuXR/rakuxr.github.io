早期アクセス — ベータ

空間や物体をスキャン。
インタラクティブな Splat を公開。
世界と共有。

Raku Capture は、スマートフォンのスキャンを AI アシスタントが対話できる Gaussian Splat に変換します — そして共有可能なリンクを提供し、誰でも閲覧・埋め込み・各自のエンジンへの取り込みができます。

無料でキャプチャを始める 公開された Splat を見る

ベータ期間中は無料 • スマートフォンまたはプロシューマーカメラ • お試しにアカウント不要

スキャンから共有可能な Splat まで 3 ステップ

キャプチャからアセットまでの基盤はベータで稼働中です。ステップ 1–3 は本日利用可能で、ステップ 3 の AI 編集ループはフェーズ 2 で深化します。

📱

1. キャプチャ

スマートフォンを持って空間や物体の周りを歩きます。Raku Capture は、スマートフォンやプロシューマーカメラが提供できるフレームを記録します — スキャン用リグも専用ソフトウェアも不要です。

⚙️

2. 再構築

当社のパイプラインがフレームを Gaussian Splat（.ply / .spz）に変換し、エンジンに適したスケールと座標系に正規化します。GPU を活用した再構築で、品質はハードウェアの可用性に応じてスケールします。

🔗

3. 公開 & 共有

クリエイタープロフィールに公開して、共有可能なリンクを取得します。誰でも Splat を閲覧・埋め込み、または Unity、Unreal、WebXR へエクスポートできます。キャプチャはあなたのもの — エクスポートは常に利用可能です。

MCP 制御レイヤー

キャプチャしたシーンに話しかける

公開された Splat はすべて MCP で操作可能なシーンでもあります。Claude、GPT-4o、または任意の MCP 互換モデルを接続し、キャプチャした空間に対して自然言語コマンドを発行できます — 17 個のネイティブツールに加え、約 9,500 個の API エンドポイントへのパススルーアクセス。

高度なシーン編集（オブジェクトの置換、再ライティング、配置/削除）はフェーズ 2 の成果物であり、本日時点では完全には出荷されていません — ただし MCP 接続と基本的なツール呼び出しは今すぐ機能します。

共有のフライホイール

公開されたキャプチャはすべて、増え続ける実世界の空間アセットのコーパスに加わります。キャプチャが増えるほど再構築の事前分布が向上し、より豊かなライブラリは埋め込みやリンクで還元する開発者を引き寄せ、リンクがより多くのクリエイターを呼び込みます。

ロードマップ — フェーズ 2

Meshy 生成ループ

フェーズ 2 では Meshy.ai のフォトリアルな家具・部屋の再デザイン生成を組み込みます。空の部屋をキャプチャし、MCP コマンドでステージングや再デザインを指示すると、Meshy が生成したアセットを AI が Splat シーンに配置し、エンジンですぐ使える形でエクスポートします。

パートナーシップの方向性は現実的で範囲も定まっています。エンドツーエンドで完全に出荷された再デザインループはフェーズ 2 の成果物であり — 現在の機能ではありません。

ロードマップ — フェーズ 3

クリエイターマーケットプレイス

フェーズ 3 では公開/利用マーケットプレイスを追加します。アセットのバージョン管理、ライセンスメタデータ、そして公開されたすべてのキャプチャの来歴情報。クリエイターへの報酬は、実際のマーケットプレイスのボリュームと GA 価格に依存します。

本日時点で報酬システムはありません。キャプチャはマーケットプレイスの状況に関わらず、常にエクスポート可能で、常にあなたのものです。

クリエイター早期アクセスに参加

キャプチャはベータ期間中無料です。Splat 出力は .ply と .spz、共有可能なプロフィールリンク、エクスポートは常に有効。早期アクセスメンバーは、フェーズ 2 の MCP 編集と Meshy 再デザインループが出荷された際に最初にアクセスできます。
