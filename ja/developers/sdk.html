SDK ダウンロード

RakuAI API の公式クライアントライブラリ。すべての SDK は開発者 NDA の対象です。

SDK のダウンロードには有効な API キーが必要です。まずは 開発者アクセスをリクエストしてください。

Python SDK

非同期サポート、型ヒント、ストリーミング、自動リトライを備えたフル機能の Python クライアント。Python 3.10 以上に対応。

バージョン: 0.1.0-beta ライセンス: プロプライエタリ サイズ: 約 45 KB
Async/Await 型ヒント ストリーミング 自動リトライ Pydantic モデル
.tar.gz をダウンロード クイックスタート
pip install raku-sdk

JavaScript / TypeScript SDK

Node.js とブラウザの両方で動作する同型 JS/TS クライアント。完全な TypeScript 型定義、SSE ストリーミング、tree-shake 対応モジュール。

バージョン: 0.1.0-beta ライセンス: プロプライエタリ サイズ: 約 32 KB
TypeScript ESM + CJS SSE ストリーミング Tree-shake 対応 ブラウザ + Node
.tgz をダウンロード クイックスタート
npm install @rakuai/sdk
