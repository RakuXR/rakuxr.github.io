よくある質問

API キーはどうやって取得できますか？ ▼
開発者 NDA に同意し、その後 ダッシュボード で API キーを作成してください。スコープ（read、generate、full）の異なる有効キーを最大 5 個まで作成できます。
どの AI クライアントがランタイムを駆動できますか？ ▼
任意の MCP 対応クライアント。現在: Claude Desktop（完全な接続ガイドと PowerShell 自動設定は /developers/claude-desktop.html）。準備中: Cursor、エージェントモードの GitHub Copilot、MCP HTTP トランスポート経由のカスタム GPT。6 つのツール（load_world_model、ingest_frame、get_scene_state、set_render_target、start_simulation、get_metrics）はすべてのクライアントで同じコントラクトです。
MCP サーフェスとは何ですか？ ▼
Model Context Protocol を介して公開される 6 つのツール: load_world_model、ingest_frame、get_scene_state、set_render_target、start_simulation、get_metrics。読み取り専用ツールはあらゆる環境で動作します。状態を変更するツールはサンドボックス／開発環境のみ、デフォルト拒否です。サーバーは型付きスキーマを強制し、完全な監査ログを書き出します。完全な設計理由は ビルドログを参照してください。
レート制限はどうなっていますか？ ▼
Free ティア：1 日あたり生成 10 回、改良 30 回。Pro ティア：1 日あたり生成 100 回、改良 500 回。Enterprise：無制限。制限超過時には HTTP 429 と Retry-After ヘッダーが返ります。レート制限のドキュメントを参照してください。
RakuAI を商用製品で使えますか？ ▼
はい。MCP サーフェス上に構築した統合は商用出荷可能です。SDK とランタイムは開発者 NDA の対象であり、Raku ランタイムエンジンの配布には有効なライセンスが必要です。ライセンスの詳細については お問い合わせください。
AI の関与はどのように開示されますか？ ▼
ランタイムはすべての MCP ツール呼び出しを監査ログに記録します: 呼び出し元、ツール、引数、タイムスタンプ、結果。アダプターマニフェストは使用中のワールドモデルバックエンド（動画予測器、ニューラル輝度場、物理ファウンデーションモデル等）を宣言します。どのアダプターがフレームを提供していても、エンジンは物理・衝突・スコアリングの決定論的な権威を保持します。

トラブルシューティング

401 Unauthorized

API キーまたは JWT トークンが欠落、期限切れ、または無効です。

Authorization: Bearer <token> ヘッダーを確認してください。必要に応じてダッシュボードからキーを再生成してください。

403 Developer Access Required

NDA 同意なしに開発者エンドポイントへアクセスしようとしています。

登録ページから開発者 NDA に同意してください。

422 Validation Error

リクエストボディに必須フィールドが欠けているか、不正な値が含まれています。または MCP ツール呼び出しがサーバーでスキーマ検証に失敗しました。

レスポンスの detail フィールドで詳細を確認してください。MCP ツール呼び出しについては、引数の型が Claude Desktop ガイドのツールスキーマと一致しているか確認してください。

429 Too Many Requests

ティアのレート制限を超えました。

Retry-After ヘッダーで再試行可能なタイミングを確認してください。より高い制限を希望する場合はティアをアップグレードしてください。

503 Service Unavailable

ランタイムまたは上流のワールドモデルアダプターが一時的に利用できません。

数秒後に再試行してください。読み取り専用ツール（get_metrics、get_scene_state）が先に復旧します。状態を変更する呼び出しの前にそれらを確認してください。
