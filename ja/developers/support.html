よくある質問

API キーはどうやって取得できますか？ ▼
開発者 NDA に同意し、その後 ダッシュボード で API キーを作成してください。スコープ（read、generate、full）の異なる有効キーを最大 5 個まで作成できます。
対応しているゲームジャンルは何ですか？ ▼
RakuAI は 13 ジャンルに対応：プラットフォーマー、スペースシューター、RPG（トップダウン、ターン制、アクション）、パズル、ビジュアルノベル、タワーディフェンス、エンドレスランナー、カードゲーム、格闘、レース、シミュレーション、サンドボックス、そして新規コンセプト向けの Custom フォールバック。
.raku 設定ファイルとは何ですか？ ▼
.raku 設定は、タイトル、ジャンル、エンティティ、シーン、レンダリング設定、オーディオ、XR 設定、AI パラメータなど、ゲームを完全に記述する JSON ドキュメントです。Raku ランタイムエンジンはこの設定を読み込んでゲームを実行します。
レート制限はどうなっていますか？ ▼
Free ティア：1 日あたり生成 10 回、改良 30 回。Pro ティア：1 日あたり生成 100 回、改良 500 回。Enterprise：無制限。制限超過時には HTTP 429 と Retry-After ヘッダーが返ります。レート制限のドキュメントを参照してください。
RakuAI を商用製品で使えますか？ ▼
はい、RakuAI で生成したゲームは商用利用できます。生成された .raku 設定はあなたのものです。ただし、SDK と API は開発者 NDA の対象であり、Raku ランタイムエンジンの配布には有効なライセンスが必要です。
生成には AI が使われていますか？どのように開示されますか？ ▼
はい、ゲーム設定は Claude (Anthropic) によって生成されます。生成されたすべての設定には ai_disclosure セクションが含まれ、どのシステムが AI を使用したか、どのガードレールが適用されたか、コンテンツがライブ生成か事前生成かが記載されています。

トラブルシューティング

401 Unauthorized

API キーまたは JWT トークンが欠落、期限切れ、または無効です。

Authorization: Bearer <token> ヘッダーを確認してください。必要に応じてダッシュボードからキーを再生成してください。

403 Developer Access Required

NDA 同意なしに開発者エンドポイントへアクセスしようとしています。

登録ページから開発者 NDA に同意してください。

422 Validation Error

リクエストボディに必須フィールドが欠けているか、不正な値が含まれています。

レスポンスの detail フィールドで詳細を確認してください。/generate では prompt が 1〜500 文字であることを確認してください。

429 Too Many Requests

ティアのレート制限を超えました。

Retry-After ヘッダーで再試行可能なタイミングを確認してください。より高い制限を希望する場合はティアをアップグレードしてください。

503 Service Unavailable

LLM サービスが一時的に利用できません（API タイムアウトまたは容量超過）。

数秒後に再試行してください。システムには自動リトライ機構（バックオフ付き 3 回）が組み込まれています。
