RakuAI 開発者向け秘密保持契約

発効日: 同意した時点

1. 定義

「機密情報」とは、RakuAI 開発者プログラムを通じて提供される非公開の技術情報、ドキュメント、API 仕様、SDK ソースコード、ゲーム設定スキーマ（.raku 形式）、および独自アルゴリズムを意味します。

2. 義務

  1. すべての機密情報を厳格に秘密として保持することに同意するものとします。
  2. RakuAI のソフトウェア、SDK、ランタイムバイナリのリバースエンジニアリング、逆コンパイル、逆アセンブルは行いません。
  3. API キー、SDK ビルド、または社内ドキュメントを、許可されていない第三者と共有しません。
  4. 開発者ツールは、RakuAI プラットフォームと統合されるアプリケーションの開発目的にのみ使用するものとします。

3. 許可された用途

  • RakuAI API を使用したアプリケーションおよびゲームの構築
  • 開発チーム内での公開ドキュメントの共有
  • RakuAI で作成したゲームの公開（適切な AI 開示の表示を含む）
  • RakuAI チームへのフィードバックおよびバグレポートの提供

4. 期間

本契約は、開発者プログラムへの参加期間中、および終了後 2 年間有効です。

5. 解約

いずれの当事者も、30 日前の書面による通知により本契約を解約できます。解約時には、所有しているすべての機密情報を破棄しなければなりません。

6. 準拠法

本契約は、アメリカ合衆国デラウェア州法に準拠します。
