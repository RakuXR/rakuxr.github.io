コンテンツパック

あらゆるテンプレートに最適化された厳選アセット — 2 つのモード、迷う余地ゼロ。

プロトタイプ vs プレゼンテーション

すべてのテンプレートに 2 種類のコンテンツパックが同梱されています。開発フェーズに合わせて選んでください。

プロトタイプモード

素早いイテレーションのために設計された軽量プレースホルダーアセット。シンプルな形状、単色、基本的なサウンドエフェクトで、アートを待たずにゲームプレイのロジックを検証できます。ゲームジャムや初期プロトタイピングに最適。

  • ✓ ファイルサイズが小さい
  • ✓ 瞬時のロード
  • ✓ 視覚的な階層が明確
  • ✓ 一貫した色分け

プレゼンテーションモード

デモ、トレーラー、ショーケース向けの磨き上げられた高品質アセット。詳細なテクスチャ、パーティクルエフェクト、アダプティブミュージック、プロフェッショナルな UI 要素。出荷品質のビジュアルがそのまま使えます。

  • ✓ HD テクスチャ・スプライト
  • ✓ 多層レイヤーのアダプティブオーディオ
  • ✓ パーティクル VFX ライブラリ
  • ✓ テーマ別 UI コンポーネント

テンプレートファミリー

各テンプレートファミリーには、プロトタイプとプレゼンテーション両方のコンテンツパックが含まれ、幅広いゲームジャンルをカバーします。

アクション・コンバット

スピード感のあるゲームプレイのためのキャラクター、環境、エフェクト、オーディオ

ストラテジー・ロジック

戦略系・パズル系ゲームに向けたマップ、UI 要素、アセット

アドベンチャー・探索

自由度の高い体験のためのワールド、タイルセット、雰囲気のあるオーディオ

カスタム

どんなジャンルのゲームでも組み立てられる柔軟なベースアセット

統合されたオーディオ

コンテンツパックには、BGM、効果音、UI 音、そしてゲームプレイにリアルタイムで反応するアダプティブな要素まで、完成されたオーディオ体験が含まれています。

コンテンツパックを利用する

コンテンツパック、アセット統合ガイド、マニフェスト仕様、完全なドキュメントは、SDK アクセスに含まれます。
