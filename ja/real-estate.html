エージェント・仲介会社・物件チーム向け

どんな物件も、探索できる 3D 空間に。

スマホで物件をスキャンするだけ — 300 ドルの機材も、専用カメラも不要 — で、RakuAI が探索可能な 3D ガウシアンスプラットのウォークスルーに再構成します。すると買い手は、動画ツアーではできなかったことができるようになります。物件に質問するのです — 「キッチンアイランドの幅は？」「主寝室はどの方角を向いている？」 — その答えは、あなたやポータルがすでに使っている任意の LLM が、当社の MCP サーフェス経由で返します。

これは RakuAI の主力バーティカルであり、私たちはこれをオープンに構築しています。以下のページは、今日出荷できるものとロードマップ上にあるものについて正直に伝えます。私たちは収益化前であり、早期アクセスのエージェントとパイロット仲介会社を探しています — 完成した製品を主張しているわけではありません。

早期アクセスを申し込む → 仕組みを見る ↓

スマホでのウォークスルーから、買い手が話しかけられる物件へ。

プラットフォーム全体を支えているのと同じ Raku Capture パイプラインを、1 ステップずつ。矢印・ドット・キーボードで操作できます。

ステップ 1

スマホでスキャン

すでにポケットに入っているスマホで物件を歩いて回るだけ。三脚も、300 ドルのキャプチャ機材も、専用カメラも、研修日も不要です。

ステップ 2

3D スプラットに再構成

フレームとモーションがクラウドで 3D ガウシアンスプラットに再構成されます — つなぎ合わせたパノラマではなく、空間のなめらかでフォトリアルに近いモデルです。

ステップ 3

買い手がブラウザで探索

リンクを 1 つ共有するだけ。買い手はどんなデバイスからでも、自分のペースで空間を歩き回れます — ヘッドセットも、アプリのインストールも、内見予約も不要です。

ステップ 4

物件に何でも質問

当社の MCP サーフェス経由で、LLM が実際のスキャンに基づいて質問に答えます — 寸法、向き、部屋に何があるか。マルチベンダー：Claude、ChatGPT、Gemini、Copilot のいずれでも駆動できます。

仕組み。

手の中のスマホから、共有でき質問できる物件まで、3 ステップ。これは RakuAI 全体で使われているのと同じ Raku Capture コアです — 不動産は出力をラップしたものであり、新しいエンジンではありません。

1

スキャン

画面のガイドに従って、スマホで住宅を歩いて回ります。キャプチャは 1 空間あたり数分で、すでに持っているもの以外は何も要りません。

2

再構成 & 公開

RakuAI がスキャンを探索可能な 3D ガウシアンスプラットに再構成し、物件情報・メール・ポータルに貼り付けられる共有リンクを 1 つお渡しします。

3

探索 & 質問

買い手はブラウザで空間を歩き回り、平易な言葉で物件に質問します — その答えは、スキャンに基づいて、MCP サーフェス経由の任意の LLM が返します。

Matterport のツアーと、どう違うのか。

3D 物件ツアー自体は新しくありません。新しいのは、それをスマホで行うこと、空間が質問に答えられるようにすること、そして決して 1 つのベンダーに縛らないことです。

一般的な 3D ツアー（例：Matterport） 不動産向け RakuAI
キャプチャ機材 多くの場合、専用カメラや約 300 ドル以上の機材、または有料のスキャン技術者の訪問が必要 すでに持っているスマホ — 専用カメラは不要
再構成 パノラマ / メッシュのウォークスルー 3D ガウシアンスプラット — よりなめらかで、よりフォトリアルに近いナビゲーション
空間への質問 ネイティブには非対応 — ビューアーであって、Q&A サーフェスではない 対応 — スキャンに基づき、MCP 経由で LLM 照会可能
どの AI が答えるか 該当なし、または単一ベンダー マルチベンダー — Claude、ChatGPT、Gemini、Copilot から選択可能
ロックイン 1 つのプラットフォームとビューアーに縛られる 中立的で標準ベースの MCP サーフェス。モデルは自由に持ち込み可能

この比較は一般的に公開されているワークフローを反映したものであり、ベンチマークではありません。他社製品の機能や価格は変わります。私たちは構築を目指している違いを示しており、まだ完成していないものについては以下で正直にお伝えします。

エージェントと仲介会社が重視する理由。

売り文句は「より凝ったツアー」ではありません。無駄な内見が減ること、買い手を事前にふるい分ける物件、そしてラグジュアリー物件だけでなくすべての物件に載せられるほど低いキャプチャコストです。

ラグジュアリーだけでなく、すべての物件に

キャプチャが有料のスキャン技術者の訪問ではなくスマホでのウォークスルーになれば、没入型ツアーはもはや高級住宅のために確保する経費項目ではなくなります。限界費用がほぼゼロに近づくので、すべての物件に載せられます。

意欲の低い内見を削減

空間を探索し、基本的な質問に事前に答えてもらえる買い手は、自ら絞り込む傾向があります。実際に行う対面の内見は、すでにバーチャルで歩いた人々とのものになります。

午前 2 時でも答える物件

「ガレージに車は 2 台入る？」「寝室はどの方角を向いている？」 そうした質問に、物件自体がスキャンに基づいて、買い手が実際に閲覧しているそのときに答えます — 営業時間中だけではありません。

あなたの AI、あなたのポータル

中立的な MCP サーフェスなので、仲介会社やポータルは物件の Q&A を、すでに料金を払っている任意の LLM に、自社ブランドで接続できます — ベンダーの強制的な乗り換えはありません。

今日できること、まだできないこと。

私たちは控えめに約束したいと考えています。RakuAI は収益化前であり、このバーティカルは初期段階です。これが、出荷済みとロードマップを分ける正直な線引きです。

今日出荷済み

  • スマホによるキャプチャとクラウドでの 3D ガウシアンスプラット再構成（Raku Capture フロー）。
  • MCP サーフェス経由のシーンクエリ — LLM が再構成された空間を巡回し、それについての質問に答えられます。
  • マルチ LLM 対応：Claude、ChatGPT、Gemini、Copilot のいずれもランタイムを駆動できます。

進行中 / ロードマップ

  • 買い手・エージェント・ポータル向けに調整された、洗練された不動産特化の物件ビューアーとワンクリック共有リンク。
  • 物件水準の Q&A テンプレート（寸法、向き、設備、「含まれるもの」）と、ポータルや MLS 隣接サイト向けの埋め込み可能ウィジェット。
  • エージェント / 仲介会社アカウント、物件単位の整理、そして買い手が何を質問するかの分析。

まだ持っていないもの

  • 有料の不動産顧客。私たちは収益化前であり、早期アクセスのエージェントとパイロット仲介会社を探しています — 持っていない契約済み顧客を主張することはありません。
  • 今日すぐ有効化できる MLS / ポータル統合。統合は現時点ではパイロットごとの作業です。
  • 計測の保証。LLM がスキャンから報告する寸法は、買い手の利便性のための見積もりであり、測量や鑑定ではありません。

早期アクセスのエージェント、またはパイロット仲介会社になりませんか。

これは私たちが賭けているバーティカルであり、実際に使う人々とともに構築しています。早期アクセス中は無料 — スキャンする実際の物件と、正直なフィードバックをお願いします。
