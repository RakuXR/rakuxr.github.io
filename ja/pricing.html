すべてのプランで利用可能

Free でも、キャプチャは LLM が読める本物のシーンです。MCP は有料プランの裏に隠していません。

MCP ランタイム、本番稼働中

Claude・ChatGPT・Gemini・Copilot を Model Context Protocol 経由でキャプチャに接続。AI はシーンを読めますが、部屋を勝手に書き換えることはありません。

スマホスキャン → 3D シーン

iPhone / Android のブラウザ PWA から撮影。サーバー側でシーンを再構成し、操作可能なスプラットと共有可能なビューアリンクが返ります。

特許に裏付けされたエンジン

5件の米国特許（優先日：2012年7月）が、すべてのキャプチャの基盤となる空間ランタイムをカバーしています。

Raku Capture は NVIDIA Inception メンバーが開発しています。NVIDIA 高速化 3D Gaussian Splatting（A10G / L40S 上の Brush トレーナー）を再構成パイプラインに導入し、スマホスキャンを数分で操作可能なスプラットに変換予定 — NGC カタログとパートナークラウドクレジットを活用します。 発表を読む →

よくある質問

Pro Beta は本当に今は無料ですか？

はい。Raku Capture がベータ期間中は Pro Beta は無料です。課金を開始する場合は、少なくとも30日前にメールでお知らせします。アップグレードしたくない場合は Free のままで構いません。正式版（GA）での予定価格は月額 $14.99 です。

「キャプチャ」とは何を指しますか？

1回のスキャンをアップロードして、3D シーンに再構成することを指します。日次カウンタは UTC 0時にリセットされます。既存のキャプチャを再表示・共有してもカウントされません。

保持期間を過ぎたキャプチャはどうなりますか？

Free ではキャプチャと再構成シーンはアップロード後7日で削除されます。Pro Beta は90日です。元の動画があれば再アップロード可能です。Enterprise はカスタム保持期間に対応します。

MCP は本当に Free にも含まれますか？

はい。MCP ランタイムは本番環境で稼働しており、Free を含む全プランで利用可能です。Claude・ChatGPT・Gemini・Copilot をキャプチャに接続すると、Model Context Protocol 経由でシーンを読み取れます。

GPU 高速化された再構成はいつ提供されますか？

サーバー側の NVIDIA A10G / L40S でのスプラットトレーニングは、NVIDIA Inception のクラウドクレジットを活用したロードマップに含まれます。現在は CPU で再構成しており、GPU が利用可能になった時点で Pro Beta と Enterprise から優先的にルーティングします。

自社製品・スマートグラス・大規模展開で使いたい。

それは Enterprise の領域です。partners@rakuai.com にユースケースと想定規模をお知らせください。ご連絡いたします。
