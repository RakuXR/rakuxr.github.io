ドキュメント

AIでゲームを作るために必要なすべて

はじめに

1

スキーマを取得

.rakuファイルスキーマがゲームフォーマットを定義します。GitHub上のサンプルゲームを参照するか、完全なスキーマについてはSDKアクセスをリクエストしてください。

2

AIにゲームを説明する

任意のAIアシスタントを開いてゲームコンセプトを説明してください。SDKにはプロンプトテンプレートとリファレンスカードが含まれており、ゲームが一発で動作します。詳細はSDKアクセスをリクエストしてください。

3

検証してプレイ

.rakuファイルを検証してプレイ。アクセスをリクエスト →

対応AIプラットフォーム

SDKアクセスをリクエスト → プロンプト、スキーマ、検証ツール

ChatGPT (OpenAI)

GPT-4およびGPT-4oは.rakuファイル生成に完全対応。最良の結果を得るために、SDK付属のプロンプトテンプレートをご使用ください。

Claude (Anthropic)

Claude OpusおよびSonnetモデルは構文的に正しい.rakuファイルを生成します。

Gemini (Google)

Gemini ProおよびUltraは.rakuスキーマに対応。マルチターン会話による反復的なゲームデザインに最適。

GitHub Copilot

VS CodeおよびVisual Studio拡張機能が.rakuファイルの補完と検証をSDKインストール時に提供します。

.rakuファイル形式
