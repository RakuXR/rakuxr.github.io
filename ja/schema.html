.rakuファイル構造

.rakuファイルは5つの必須セクションを持つJSONドキュメントです。

1

raku_format

スキーマバージョン文字列。現在は"1.0.0"。

2

metadata

ゲーム情報：名前、説明、ジャンル（space_shooter、tower_defense、puzzle、platformer、arena_combatなど）、作成者、バージョン、タグ。

3

initialization

エンジンのセットアップ呼び出し。raku_initで開始する必要があります。ウィンドウ、物理、オーディオ、AIサブシステムを設定します。

4

calls

ゲームプレイのセットアップ：シーンの作成、アセットの読み込み、入力のバインド、AIの設定。raku_scene_start_loopで終了する必要があります。

5

shutdown

クリーンアップ：シーンの破棄とエンジンのシャットダウン。raku_shutdownで終了する必要があります。

16のエンジンサブシステム

コア

raku_init、raku_shutdown

レンダラー

raku_renderer_create_window、raku_renderer_create_camera、raku_renderer_create_light

シーン

raku_scene_create、raku_scene_create_sprite、raku_scene_start_loop

物理

raku_physics_create_world、raku_physics_add_rigidbody

オーディオ

raku_audio_init、raku_audio_play、raku_audio_play_spatial

AI & SLM

raku_ai_set_difficulty、raku_slm_init、raku_ai_create_agent

入力

raku_input_bind_key

XR

raku_xr_init — OpenXR 1.1、ハンド/アイトラッキング

アニメーション

raku_animation_load、raku_animation_play

ネットワーク

raku_network_init — LAN、WebRTC、レプリケーション

UI

raku_ui_create_label — 2D/3Dウィジェット、HUD

アセット

raku_assets_load_model — GLTF、FBX、ストリーミング

AIプロンプト例

このプロンプトをChatGPT、Claude、または任意のAIアシスタントにコピーしてください：

// Paste this prompt into any AI assistant:

Generate a .raku game file for a space shooter game with these features:
- Adaptive difficulty that scales with player performance
- Keyboard controls (arrows + space to fire)
- Background music with spatial audio
- 1280x720 window

Use the Raku Runtime schema from https://rakuai.com/schema.json
The file must include: raku_format, metadata, initialization, calls, shutdown.
All API functions follow the pattern: raku_<module>_<action>
