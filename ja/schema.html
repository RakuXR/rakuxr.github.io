.rakuファイル構造

.rakuファイルは5つの必須セクションを持つJSONドキュメントです。

1

raku_format

スキーマバージョン文字列。現在は"1.0.0"。

2

metadata

ゲーム情報：名前、説明、ジャンル（space_shooter、tower_defense、puzzle、platformer、arena_combatなど）、作成者、バージョン、タグ。

3

initialization

エンジンのセットアップ呼び出し。ウィンドウ、物理、オーディオ、AIサブシステムを設定します。

4

calls

ゲームプレイのセットアップ：シーンの作成、アセットの読み込み、入力のバインド、AIの設定。

5

shutdown

クリーンアップ：シーンの破棄とエンジンのシャットダウン。

エンジンサブシステム

コア

エンジンライフサイクル、メモリ管理、スレッディング

レンダラー

ウィンドウ管理、カメラ、ライティング、PBRマテリアル

シーン

シーングラフ、エンティティ管理、ゲームループ

物理

剛体シミュレーション、衝突検出、レイキャスティング

オーディオ

空間オーディオ、HRTF、適応型ミュージック

AI & SLM

動的難易度調整、プレイヤープロファイリング、オンデバイスML

入力

キーボード、マウス、ゲームパッド、タッチ、XRコントローラー

XR

OpenXR 1.1、ハンド/アイトラッキング

アニメーション

スケルタルアニメーション、ブレンドツリー、IKソルバー

ネットワーク

LAN、WebRTC、レプリケーション

UI

2D/3Dウィジェット、HUD

アセット

GLTF、FBX、ストリーミング

AIプロンプト例

このプロンプトをChatGPT、Claude、または任意のAIアシスタントにコピーしてください：

// Paste this prompt into any AI assistant:

Generate a .raku game file for a space shooter game with these features:
- Adaptive difficulty that scales with player performance
- Keyboard controls (arrows + space to fire)
- Background music with spatial audio
- 1280x720 window

Use the Raku Runtime schema from https://rakuai.com/schema.json
The file must include: raku_format, metadata, initialization, calls, shutdown.
