ランタイムスキーマ & MCP コントラクト

任意の AI アシスタントが RakuAI ランタイムを駆動するために使用する標準コントラクト — 6 つの MCP ツール、アダプターマニフェスト、シーン状態スナップショット。

AI アシスタント・エージェント向け

6 つの MCP ツール

MCP サーバーは固定されたサーフェスを公開します — load_world_model、ingest_frame、get_scene_state、set_render_target、start_simulation、get_metrics。読み取り専用ツールはすべてのモードで動作し、状態を変更するツールはサンドボックス／開発環境のみ。

MCP ビルドログを読む →

標準コントラクト、任意のエージェント

Claude Desktop、Cursor、カスタム GPT、その他任意の MCP 対応クライアントからランタイムを駆動。ツールサーフェスは同じです。Claude Desktop の接続ガイドは現在公開中。他のクライアントも準備中です。

Claude Desktop を接続 →

API リファレンス

完全なサーバー、アダプター、シーン状態スキーマは NDA 下で利用可能です。NDA アクセスをリクエストして、完全な統合キットを受け取ってください。

MCP サーフェス

外部エージェントはすべて同じ 6 つのツールを通じてエンジンに到達します。デフォルト拒否のパーミッション、型付きスキーマ、完全な監査ログ。サーバーは境界です — エンジンは物理・衝突・スコアリング・マルチプレイヤー状態の権威を保持します。

MCP ドキュメントを読む

エンジンサブシステム

Raku エンジンはレンダリング、物理、オーディオ、AI、XR、ネットワーキング、アニメーションなどをカバーするモジュラーアーキテクチャです。アダプターは MCP 境界経由でプラグインし、エンジンは決定論的な権威を保持します。

完全な API ドキュメントとサブシステムの詳細は NDA 下で利用可能です。アクセスをリクエスト →

AI アシスタントを接続

RakuAI MCP サーバーを Claude Desktop、Cursor、または任意の MCP 対応クライアントに追加してください。6 つのツールが会話の中でネイティブに表示されます。すでに使用しているアシスタントからランタイムを駆動。
