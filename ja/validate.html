ランタイムコントラクトを検証

MCP ツール呼び出し、アダプターマニフェスト、シーン状態スナップショットのスキーマ検証。

検証ツール準備中

6 つのツールから成る MCP サーフェス（load_world_model、ingest_frame、 get_scene_state、set_render_target、start_simulation、 get_metrics）には、サーバーで強制される型付きスキーマがあります。アダプターマニフェストと シーン状態スナップショットのためのスタンドアロンな貼り付け検証ツールは近日公開予定です。 現在は、ライブサーバーがすべての呼び出しを検証し、精密なエラーを返します。

MCP ビルドログを読む Claude Desktop を接続

AI アシスタントを接続

RakuAI MCP サーバーを Claude Desktop、Cursor、または任意の MCP 対応クライアントに追加してください。6 つのツールが 会話の中でネイティブに表示され、エンジンがワールドモデルに触れる前に、サーバーがすべての呼び出しを検証します。
