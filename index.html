How It Works

1

Describe Your Game

Tell any AI assistant what game you want. "A space shooter with adaptive difficulty and voice commands."

2

The AI generates a complete game definition in JSON format. All game logic, AI systems, and personalization settings included.

3

Double-click the .raku file. The runtime loads your game in under 1 second. No Unity, no Unreal, no build step.

Why It is Different

AI-Native Architecture

Dynamic difficulty, player profiling, adaptive audio, and NPC personality are built into the engine core. Not a plugin. Always on.

Instant Execution

.raku files are JSON declarations, not executables. The runtime interprets them in real-time. Edit the JSON, refresh the game. No compile.

On-Device AI

Small Language Models run locally (RakuSLM). No cloud calls. Privacy-first. Works offline. Personalization happens instantly.

XR-Ready

OpenXR 1.1, hand/eye tracking, spatial anchors, and passthrough. Every game is XR-capable by default. Toggle with one flag.

Spatial Audio + HRTF

Positional audio with head-related transfer functions. Adaptive music system responds to gameplay intensity. Voice assistant integration.

Open Specification

.raku file format is documented. Parse it, mod it, generate it from any tool. Runtime is free. No license fees. No royalties.
