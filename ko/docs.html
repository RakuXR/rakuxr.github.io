문서

현실 세계에서 AI를 강화하세요.

17개 네이티브 MCP 도구 표면으로 어떤 AI 어시스턴트로든 RakuAI 런타임을 구동하세요. 18ms 모션-투-포톤. 서브 밀리미터 공간 정밀도. 노트북이 아닌 안경을 위해 만들어졌습니다.

왜 RakuAI인가?

미국 등록 특허 5건. 우선일 2012년 7월. 오늘날의 AR 안경 카테고리를 뒷받침하는 10년 묵은 IP. Claude, ChatGPT, Gemini, Copilot이 모두 구동하는 런타임 — 설계상 멀티 벤더.

노트북이 아닌 안경을 위해 만들어짐

18ms 모션-투-포톤 지연. 서브 밀리미터 공간 정밀도. Meta Quest 3, Xreal Air, Samsung Android XR, MiRZA AR2에서 OpenXR 적합.

월드 모델 오케스트레이션

월드 모델 오케스트레이션을 위한 17개 네이티브 MCP 도구 표면. 매 프레임, 모든 NPC, 모든 상호작용이 당신이 선택한 AI를 통해 라우팅됩니다. Unity의 MCP는 디자인 타임에 장면 조작을 대상으로 하지만 — RakuAI의 MCP는 런타임에, 매 프레임마다 실행됩니다.

산업 등급 표면

네이티브 DLL 18개. 라우터 모듈 17개. 9,500+ REST 엔드포인트. 바인딩된 엔진 함수 10,328개. 재미있게 느껴지는 런타임 뒤에 있는 지루한 숫자들.

2

런타임 구동

모든 환경에서 읽기 전용 검사를 위해 get_metrics 또는 get_scene_state를 호출하세요. 샌드박스/개발 환경에서는 어댑터를 로드하고, 프레임을 수집하고, 렌더 타깃을 구성하고, 시뮬레이션을 시작하세요. 엔진은 물리, 충돌, 점수 산정에 대한 권한을 유지합니다.

17개 네이티브 MCP 도구 표면

모든 외부 에이전트는 동일한 17개 네이티브 도구를 통해 엔진에 도달합니다 — get_scene_state, add_scene_node, load_world_model, … (읽기 전용 5개 + 변경 12개; mcp.html 참조). 표준 계약, 모든 에이전트.

지원되는 AI 클라이언트

Claude Desktop 연결하기 → 자동 구성 스크립트와 전체 도구 레퍼런스를 확인하세요

Claude Desktop

네이티브 MCP 지원. 개발자 포털의 PowerShell 자동 구성 스크립트; 모든 플랫폼을 위한 수동 JSON 안내.

Cursor & IDE 클라이언트

MCP 지원 에디터나 IDE라면 무엇이든 동일한 17개 네이티브 도구 표면을 통해 런타임에 도달할 수 있습니다. 연결 가이드는 곧 제공됩니다.

커스텀 에이전트

MCP 경계에서 직접 에이전트를 구축하세요. 읽기 전용 도구는 프로덕션에서 작동하며; 변경 도구는 샌드박스/개발 전용이고 기본 거부입니다.

자체 LLM 사용

MCP 표면은 모델 비의존적입니다. ChatGPT 커스텀 GPT, Gemini, Llama, 사내 모델 모두 동일한 계약을 사용합니다.

아키텍처 개요

엔진이 결정론적 권한입니다. 월드 모델 — 비디오 예측기, 뉴럴 래디언스 필드, 물리 파운데이션 모델 — 은 MCP 어댑터를 통해 연결되는 기여자입니다. 이들 중 어느 것도 시뮬레이션에서 실제로 일어나는 일에 대한 엔진의 권한을 무시하지 않습니다.
