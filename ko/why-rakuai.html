AI 네이티브 공간 엔진

NPC가 말을 걸고, 헤드셋이 주류가 되며, 브라우저가 일급 런타임이 된 세상을 위해 설계된 새로운 카테고리의 게임 엔진입니다. 단일 API. 세 가지 타겟. 만 가지의 기능.

API 둘러보기 엔진 라이브로 보기

다른 어떤 엔진도 제공하지 않는 세 가지

Unity, Unreal, Godot은 ChatGPT 이전, Quest Pro 이전, WebGPU 이전에 설계되었습니다. RakuAI는 처음부터 이 세 가지 모두를 염두에 두고 설계되었습니다.

☷
하나의 코드베이스

어디서나 같은 API

브라우저에서, 네이티브 Windows 클라이언트에서, 또는 XR 헤드셋에서 동일한 REST 엔드포인트를 호출하세요. 별도의 WebGL 포트도, 축소된 모바일 SKU도, 플랫폼별 함정도 없습니다. 한 번 만들고 웹, 네이티브, 몰입형 전반에 동시에 출시하세요.

☻
에이전트 우선

AI 네이티브 엔진

LLM 기반 NPC, 온디바이스 소형 언어 모델 추론, 감정 모델링, 대화 트리가 나중에 덧붙인 플러그인이 아니라 일급 서브시스템입니다. 엔진은 보고, 생각하고, 말할 수 있는 에이전트를 중심으로 설계되었습니다.

⚙
프로덕션 스케일

10,000+ 함수

네이티브 서브시스템 18개. 엔진 함수 10,328개. REST 엔드포인트 9,500+개. 물리, 렌더링, XR, 오디오, 애니메이션, 네트워킹, 스크립팅 — 프레임워크나 데모가 아닌 진짜 엔진의 전체 표면적입니다. 실전에서 검증된, 프로덕션 워크로드에 바로 투입할 수 있는 엔진입니다.

Unity가 아닙니다. Unreal이 아닙니다. Godot이 아닙니다.
새로운 것입니다.

기존 엔진들은 네이티브 바이너리로 렌더링되는 싱글플레이어 데스크톱 게임의 시대를 위해 만들어졌습니다. RakuAI는 모든 캐릭터가 말하고, 모든 헤드셋에 핸드 트래킹이 기본 탑재되며, 모든 게임이 브라우저 탭에서 링크 하나로 열리는 세상을 위해 만들어졌습니다. 기반 자체가 달라야 했습니다.
