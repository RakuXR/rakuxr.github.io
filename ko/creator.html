얼리 액세스 — 베타

공간이나 사물을 스캔하세요.
인터랙티브 스플랫을 게시하세요.
전 세계와 공유하세요.

Raku Capture는 휴대폰 스캔을 AI 어시스턴트가 대화할 수 있는 가우시안 스플랫으로 변환하고, 누구나 보고 임베드하거나 자신의 엔진에 넣을 수 있는 공유 링크를 제공합니다.

무료로 캡처 시작 게시된 스플랫 둘러보기

베타 기간 무료 • 휴대폰 또는 프로슈머 카메라 • 체험에 계정 불필요

스캔에서 공유 가능한 스플랫까지 세 단계

캡처-투-에셋 기반은 베타로 가동 중입니다. 1–3단계는 현재 작동하며, 3단계의 AI 편집 루프는 Phase 2에서 심화됩니다.

📱

1. 캡처

공간이나 사물 주위로 휴대폰을 움직이세요. Raku Capture는 휴대폰이나 프로슈머 카메라가 제공하는 프레임을 기록합니다 — 스캐닝 장비도, 전문 소프트웨어도 필요 없습니다.

⚙️

2. 재구성

당사의 파이프라인은 프레임을 가우시안 스플랫(.ply / .spz)으로 변환하고, 엔진에 적합한 스케일과 좌표계로 정규화합니다. GPU 기반 재구성이며, 품질은 하드웨어 가용성에 따라 확장됩니다.

🔗

3. 게시 & 공유

크리에이터 프로필에 게시하고 공유 링크를 받으세요. 누구나 스플랫을 보고, 임베드하거나, Unity, Unreal, WebXR로 내보낼 수 있습니다. 캡처는 당신의 것입니다 — 내보내기는 항상 가능합니다.

MCP 제어 레이어

캡처한 장면과 대화하세요

게시된 모든 스플랫은 MCP로 구동 가능한 장면이기도 합니다. Claude, GPT-4o 또는 MCP 호환 모델을 연결하고 캡처한 공간에 자연어 명령을 내리세요 — 17개의 네이티브 도구와 약 9,500개 API 엔드포인트로의 패스스루 액세스.

심층 장면 편집(객체 교체, 리라이트, 배치/제거)은 Phase 2 결과물로, 현재 완전히 출시되지는 않았습니다 — 하지만 MCP 연결과 기본 도구 호출은 지금 작동합니다.

공유 플라이휠

게시된 모든 캡처는 점점 늘어나는 실제 공간 에셋 코퍼스에 더해집니다. 캡처가 많아지면 재구성 사전 정보가 향상되고, 풍부한 라이브러리는 임베드하고 링크하는 개발자를 끌어들이며, 링크는 더 많은 크리에이터를 데려옵니다.

로드맵 — Phase 2

Meshy 생성 루프

Phase 2 wires in Meshy.ai의 포토리얼 가구 및 룸 리디자인 생성을 연결합니다. 빈 방을 캡처하고, MCP 명령으로 스테이징하거나 리디자인하면, Meshy가 생성한 에셋을 AI가 스플랫 장면에 배치하고 엔진에 바로 쓸 수 있도록 내보냅니다.

파트너십 방향은 실재하며 범위가 정해져 있습니다. 완전히 출시된 엔드-투-엔드 리디자인 루프는 Phase 2 결과물이며 — 현재 기능이 아닙니다.

로드맵 — Phase 3

크리에이터 마켓플레이스

Phase 3은 게시/소비 마켓플레이스를 추가합니다: 에셋 버전 관리, 라이선싱 메타데이터, 그리고 게시된 모든 캡처의 출처 정보. 크리에이터 정산은 실제 마켓플레이스 거래량과 GA 가격 책정에 따라 결정됩니다.

현재 정산 시스템은 없습니다. 캡처는 마켓플레이스 상태와 관계없이 항상 내보낼 수 있으며 항상 당신의 것입니다.

크리에이터 얼리 액세스에 참여하세요

캡처는 베타 기간 동안 무료입니다. .ply 및 .spz 스플랫 출력, 공유 가능한 프로필 링크, 항상 켜진 내보내기. 얼리 액세스 멤버는 Phase 2 MCP 편집과 Meshy 리디자인 루프가 출시될 때 우선 액세스를 받습니다.
