XR 플랫폼

완전한 XR 개발 플랫폼

포괄적인 XR 입력 및 상호작용 지원. 광범위한 헤드셋 호환성을 위해 업계 표준 위에 구축되었습니다.

Meta Quest 3 Apple Vision Pro PlayStation VR2 SteamVR 더 많은 헤드셋
상호작용

자연스러운 입력

직관적이고 컨트롤러가 필요 없는 XR 경험을 위한 다양한 입력 방식. 핸드 트래킹, 시선 상호작용, 컨트롤러 지원을 모든 플랫폼에 걸쳐 통합된 추상화로 제공합니다.

핵심 이점: 하나의 입력 API. 모든 헤드셋. 플랫폼별 입력 코드가 필요 없습니다.
피드백

풍부한 햅틱 피드백

컨트롤러와 손 전반에 걸친 다중 감각 피드백. 표면, 충돌, UI 신호에 대한 맥락적 햅틱 패턴이 플레이어를 경험 속으로 끌어들입니다.

핵심 이점: 더 깊은 몰입. 플레이어는 게임을 보는 것을 넘어 느낍니다.
공간

혼합 현실 & 공간 앵커

패스스루, 장면 이해, 지속적인 월드 고정 콘텐츠. 플레이어 주변의 현실 세계를 이해하고 그것과 융합되는 게임.

핵심 이점: 세션과 사용자를 넘어 유지되는 지속적이고 월드 인식형 경험.
성능

XR 최적화 렌더링

모든 서브시스템이 XR 프레임 예산에 맞게 튜닝되어 있습니다. 포비티드 렌더링 지원, 리프로젝션, 자동 LOD 관리를 통한 일관된 72-120 Hz 렌더링.

핵심 이점: 프레임 드롭이나 멀미 없는 부드럽고 편안한 VR.

하나의 API. 모든 헤드셋.

Raku는 플랫폼 차이를 추상화하여 XR 코드가 어디서나 실행되도록 합니다. 한 번 작성하고 모든 주요 헤드셋에 배포하세요.

크로스 플랫폼

앱이 지원되는 모든 헤드셋에서 자동으로 훌륭하게 작동합니다.

통합 입력

모든 플랫폼에 걸친 하나의 입력 시스템.

성능 최적화

각 기기에 대한 자동 성능 최적화.
