엔터프라이즈 공간 AI

원패스 공간 캡처. 모든 LLM이 호출할 수 있는 MCP 런타임 — 17개 네이티브 도구, 약 147개 패스스루 도구, 약 9,500개 API 엔드포인트. 엔터프라이즈 수직 통합을 위해 구축: 보험(클레임 & 내용물 목록), 리테일 & 시설(공간 계획, 변경 감지), 로보틱스(영속적 공간 메모리) — 무엇이 GA이고 무엇이 로드맵 단계인지 솔직하게 표기합니다.

엔터프라이즈 수직 통합

세 가지 버티컬, 하나의 공간 캡처 + MCP 기반. 보험과 로보틱스는 자금이 투입된 활성 웨지입니다. 리테일 & 시설은 수요 기반으로 대응하며 — 디자인 파트너가 확정되면 자금이 투입된 웨지로 승격됩니다.

세 가지 모두 동일한 Raku Capture 파이프라인 + MCP 도구 서피스에서 실행됩니다: api.rakuai.com.

확장

18ms 목표지원되는 하드웨어 전반에서 모션-투-포톤 목표를 유지합니다.
발열 인식동적 품질 폴백으로 디바이스를 동작 한계 내로 유지합니다.

보안 & 거버넌스

기본 거부정책이 명시적으로 허용하지 않는 한 아무것도 실행되지 않습니다.
RBAC + SSO/SCIM엔터프라이즈 ID 통합을 갖춘 역할 기반 액세스.
불변 감사컴플라이언스와 인시던트 검토를 위해 모든 결정을 로깅합니다.

엔터프라이즈 프리미티브

모든 공간 캡처 버티컬에 필요한 공유 플랫폼 — 데이터 레지던시, SSO, 프라이빗/브랜드 MCP 엔드포인트, OEM은 제한된 로드맵 항목이며 GA가 아닙니다. 캡처 + 쿼리는 현재 사용 가능합니다. 거버넌스 & ID, 커스텀 MCP / OEM, 데이터 레지던시는 단계별로 명확히 표기됩니다.

버티컬과 사용 사례를 알려주시면 — 맞춤형 파일럿 제안을 준비하겠습니다. 파일럿 요청 ↓

캡처 + 쿼리  GA

쿼리 가능한 공간 모델로의 원패스 휴대폰 캡처. 모든 LLM이 결과에 대해 MCP 도구 서피스를 호출할 수 있습니다.

  • 휴대폰 기반 스플랫 캡처
  • MCP 도구 서피스 (쿼리 / 실행)
  • 버티컬 템플릿: 보험, 리테일, 로보틱스
  • 프로젝트별 공간 범위 지정
  • API 키 / 토큰 인증 (GA)

거버넌스 & ID  Roadmap

엔터프라이즈 조달에 필요한 인증, 감사, 플릿 정책 레이어. Phase 1 빌드 단계 — 아직 프로덕션이 아닙니다.

  • SSO (OIDC / SAML) — Phase 1
  • SCIM 프로비저닝 — Phase 1
  • 기본 거부 RBAC — Phase 1
  • AI 작업 감사 로그 — Phase 1
  • SOC 2 준비 평가 — 진행 중

커스텀 MCP & OEM  Roadmap

테넌트별 커스텀 MCP 네임스페이스(귀사의 시스템을 당사 시스템과 함께 호출 가능) 및 프라이빗 / 화이트라벨 엔드포인트. Phase 2 목표.

  • 테넌트별 MCP 도구 네임스페이스 — Phase 2
  • 프라이빗 / 브랜드 MCP 엔드포인트 — Phase 2
  • 데이터 레지던시 옵션 — Phase 3
  • 커스텀 브랜딩 / OEM 번들 — Phase 2
  • 계약상 SLA — Phase 2

Phase 1 = 진행 중인 빌드. Phase 2–3 = 자금이 투입된 로드맵이며 아직 GA가 아님. 아직 SOC 2 보고서 없음; 준비 평가 진행 중.

솔직한 현황: 캡처 + 쿼리는 프로덕션 상태입니다. SSO/SCIM, 기본 거부 RBAC, 감사 로깅, 커스텀 MCP 네임스페이스, 프라이빗/브랜드 엔드포인트, 데이터 레지던시, OEM 번들링은 모두 아직 GA가 아닌 로드맵 항목입니다. 오늘 SOC 2를 요청하는 조달 담당자는 솔직하게 “준비 평가 진행 중”이라는 답변을 받습니다. 조작된 고객도, 온콜로 뒷받침되지 않는 SLA도 없습니다. 문의하시면 무엇이 사용 가능한지 정확히 알려드립니다.
