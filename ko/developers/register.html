RakuAI 개발자 비밀 유지 계약 (NDA)

효력 발생일: 동의 시점

1. 정의

"기밀 정보"란 RakuAI 개발자 프로그램을 통해 제공되는 모든 비공개 기술 정보, 문서, API 사양, SDK 소스 코드, 게임 설정 스키마(.raku 형식), 독점 알고리즘을 의미합니다.

2. 의무

  1. 귀하는 모든 기밀 정보를 엄격히 비밀로 유지하는 데 동의합니다.
  2. 귀하는 RakuAI 소프트웨어, SDK, 런타임 바이너리에 대해 리버스 엔지니어링, 디컴파일, 디스어셈블을 수행하지 않습니다.
  3. 귀하는 API 키, SDK 빌드, 내부 문서를 권한이 없는 자와 공유하지 않습니다.
  4. 귀하는 개발자 도구를 오직 RakuAI 플랫폼과 통합되는 애플리케이션 개발 목적으로만 사용합니다.

3. 허용되는 사용

  • RakuAI API를 사용하여 애플리케이션 및 게임 개발
  • 개발팀과 공개 문서 공유
  • RakuAI로 제작한 게임의 퍼블리싱 (적절한 AI 공지 포함)
  • RakuAI 팀에 피드백과 버그 보고서 제공

4. 기간

본 계약은 개발자 프로그램 참여 기간 동안, 그리고 종료 후 2년간 효력을 유지합니다.

5. 해지

각 당사자는 30일 전 서면 통지로 본 계약을 해지할 수 있습니다. 해지 시 귀하는 보유 중인 모든 기밀 정보를 폐기해야 합니다.

6. 준거법

본 계약은 미국 델라웨어주 법률에 따릅니다.
