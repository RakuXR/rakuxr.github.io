Validate the Runtime Contract

Supercharge your AI in the real world.

Schema validation for MCP tool calls, adapter manifests, and scene-state snapshots. The runtime that Claude, ChatGPT, Gemini, and Copilot all drive.

Validation Tooling in Flight

The six-tool MCP surface (load_world_model, ingest_frame, get_scene_state, set_render_target, start_simulation, get_metrics) has typed schemas enforced at the server. The standalone paste-and-validate tool for adapter manifests and scene-state snapshots is shipping next. For now, the live server validates every call and returns precise errors.
