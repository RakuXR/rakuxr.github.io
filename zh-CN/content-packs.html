内容包

为每个模板精选的素材 — 两种模式，无需猜测。

原型 vs 演示

每个模板都附带两个内容包。选择适合你当前开发阶段的那一个。

原型模式

专为快速迭代设计的轻量占位素材。简单的形状、纯色与基础音效，让你无需等待美术资源即可测试玩法逻辑。非常适合 Game Jam 与早期原型开发。

  • ✓ 文件体积极小
  • ✓ 即时加载
  • ✓ 清晰的视觉层级
  • ✓ 一致的配色编码

演示模式

面向 Demo、预告片与展示的精致高保真素材。细致的贴图、粒子特效、自适应音乐与专业 UI 元素。开箱即用、可发布级别的画面。

  • ✓ 高清贴图与精灵
  • ✓ 多层自适应音频
  • ✓ 粒子特效库
  • ✓ 主题化 UI 组件

模板系列

每个模板系列都同时包含原型与演示内容包，覆盖广泛的游戏类型。

策略与逻辑

冒险与探索
