Raku Capture 将手机扫描转化为 AI 助手可以对话的高斯splat — 然后为你提供可分享链接，任何人都能查看、嵌入或将其放入自己的引擎。

测试期免费 • 手机或专业级相机 • 无需账户即可试用

三步从扫描到可分享的splat

捕获到资产的基础功能已在测试版上线。第 1–3 步当前可用；第 3 步的 AI 编辑循环将在第 2 阶段深化。

📱

1. 捕获

用手机环绕空间或物体移动。Raku Capture 记录你的手机或专业级相机所能提供的帧 — 无需扫描设备，无需专业软件。

⚙️

2. 重建

我们的流水线将你的帧转换为高斯splat（.ply / .spz），并归一化为适合引擎的比例与坐标系。基于 GPU 的重建；质量随硬件可用性而提升。

🔗

3. 发布与分享

发布到你的创作者资料并获取可分享链接。任何人都能查看 splat、嵌入或将其导出到 Unity、Unreal 或 WebXR。你的捕获归你所有 — 始终可导出。

MCP 控制层

与你捕获的场景对话

每个已发布的 splat 也是可由 MCP 驱动的场景。连接 Claude、GPT-4o 或任何兼容 MCP 的模型，对你捕获的空间发出自然语言命令 — 17 个原生工具加上对约 9,500 个 API 端点的透传访问。

深度场景编辑（替换物体、重新打光、放置/移除）是第 2 阶段的交付内容，目前尚未完全发布 — 但 MCP 连接和基本工具调用现已可用。

路线图 — 第 2 阶段

Meshy 生成循环

Phase 2 wires in Meshy.ai 的照片级家具与房间重设计生成。捕获一个空房间；发出 MCP 命令进行布置或重设计；Meshy 生成的资产由 AI 放入 splat 场景，并导出为可直接用于引擎的格式。

合作方向真实且范围明确。完全发布的端到端重设计循环是第 2 阶段的交付内容 — 而非当前功能。

路线图 — 第 3 阶段

创作者市场
