RakuAI 开发者保密协议（NDA）

生效日期：自接受之日起

1. 定义

“保密信息”指通过 RakuAI 开发者计划提供的任何非公开技术信息、文档、API 规范、SDK 源代码、游戏配置模式（.raku 格式）以及专有算法。

2. 义务

  1. 你同意对所有保密信息严格保密。
  2. 你不得对任何 RakuAI 软件、SDK 或运行时二进制文件进行逆向工程、反编译或反汇编。
  3. 你不得向未获授权的第三方分享 API 密钥、SDK 构建或内部文档。
  4. 你将仅出于开发与 RakuAI 平台集成的应用之目的使用开发者工具。

3. 允许的用途

  • 使用 RakuAI API 构建应用与游戏
  • 与你的开发团队分享公开文档
  • 发布用 RakuAI 创建的游戏（须做适当的 AI 披露）
  • 向 RakuAI 团队提供反馈与 Bug 报告

4. 有效期

本协议在你参与开发者计划期间持续有效，并在终止后再延续两（2）年。

5. 终止

任何一方均可提前 30 天书面通知终止本协议。终止后，你必须销毁你持有的所有保密信息。

6. 适用法律
