Governance & Identity  Roadmap

The auth, audit, and fleet-policy layer required for enterprise procurement. In Phase 1 build — not yet in production.

  • SSO (OIDC / SAML) — Phase 1
  • SCIM provisioning — Phase 1
  • Deny-by-default RBAC — Phase 1

Custom MCP & OEM  Roadmap

Per-tenant custom MCP namespaces (your systems callable alongside ours) and private / white-label endpoints. Phase 2 target.

  • Per-tenant MCP tool namespaces — Phase 2
  • Private / branded MCP endpoint — Phase 2
  • Data residency options — Phase 3
  • Custom branding / OEM bundle — Phase 2
  • Contractual SLAs — Phase 2

Phase 1 = active build. Phase 2–3 = funded roadmap, not yet GA. No SOC 2 report yet; readiness assessment in progress.
