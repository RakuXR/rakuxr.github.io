Acuerdo de Confidencialidad para Desarrolladores de RakuAI

Fecha de vigencia: Al aceptar

1. Definiciones

"Información Confidencial" significa cualquier información técnica no pública, documentación, especificaciones de API, código fuente del SDK, esquemas de configuración de juegos (formato .raku) y algoritmos propietarios proporcionados a través del Programa de Desarrolladores de RakuAI.

2. Obligaciones

  1. Aceptas mantener toda la Información Confidencial estrictamente confidencial.
  2. No realizarás ingeniería inversa, descompilarás ni desensamblarás ningún software, SDK o binario de runtime de RakuAI.
  3. No compartirás claves API, builds del SDK ni documentación interna con terceros no autorizados.
  4. Usarás las herramientas para desarrolladores únicamente para desarrollar aplicaciones que se integren con la plataforma RakuAI.

3. Usos permitidos

  • Construir aplicaciones y juegos usando la API de RakuAI
  • Compartir documentación pública con tu equipo de desarrollo
  • Publicar juegos creados con RakuAI (con la divulgación adecuada de uso de AI)
  • Proporcionar retroalimentación y reportes de errores al equipo de RakuAI

4. Duración

Este acuerdo permanece vigente durante tu participación en el Programa de Desarrolladores y por dos (2) años después de su terminación.

5. Terminación

Cualquiera de las partes puede terminar este acuerdo con un aviso por escrito de 30 días. Al terminar, debes destruir toda la Información Confidencial en tu poder.

6. Ley aplicable

Este acuerdo se rige por las leyes del Estado de Delaware, Estados Unidos.
