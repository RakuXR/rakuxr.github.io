Paquetes de contenido

Assets curados para cada plantilla — dos modos, cero conjeturas.

Prototipo vs Presentación

Cada plantilla viene con dos paquetes de contenido. Elige el que se ajuste a tu etapa de desarrollo.

Modo Prototipo

Assets de marcador de posición ligeros, diseñados para iterar rápido. Formas simples, colores sólidos y efectos de sonido básicos para que puedas probar la lógica de juego sin esperar al arte. Ideal para game jams y prototipado temprano.

  • ✓ Tamaños de archivo mínimos
  • ✓ Tiempos de carga instantáneos
  • ✓ Jerarquía visual clara
  • ✓ Codificación de color consistente

Modo Presentación

Assets pulidos de alta fidelidad para demos, tráileres y showcases. Texturas detalladas, efectos de partículas, música adaptativa y elementos de UI profesionales. Visuales con calidad de producción listos para usar.

  • ✓ Texturas y sprites HD
  • ✓ Audio adaptativo multicapa
  • ✓ Biblioteca de VFX de partículas
  • ✓ Componentes de UI temáticos

Familias de plantillas

Cada familia de plantillas incluye paquetes de contenido tanto de Prototipo como de Presentación, cubriendo una amplia gama de géneros de juego.

Acción y combate

Personajes, entornos, efectos y audio para jugabilidad de ritmo rápido

Estrategia y lógica

Mapas, elementos de UI y assets para juegos estratégicos y de puzzles

Aventura y exploración

Mundos, tilesets y audio ambiental para experiencias abiertas

Personalizado

Assets base flexibles para construir cualquier género de juego

Audio integrado

Los paquetes de contenido incluyen una experiencia de audio completa con música de fondo, efectos de sonido, audio de UI y elementos adaptativos que responden a la jugabilidad en tiempo real.

Acceder a los paquetes de contenido

Los paquetes de contenido, las guías de integración de assets, las especificaciones de manifiesto y la documentación completa se incluyen con el acceso al SDK.
