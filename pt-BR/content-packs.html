Pacotes de Conteúdo

Assets Selecionados para Cada Template — dois modos, zero adivinhação.

Protótipo vs Apresentação

Cada template vem com dois pacotes de conteúdo. Escolha o que combina com o seu estágio de desenvolvimento.

Modo Protótipo

Assets leves de marcação projetados para iteração rápida. Formas simples, cores sólidas e efeitos sonoros básicos para que você teste a lógica de gameplay sem esperar pela arte. Ideal para game jams e prototipagem inicial.

  • ✓ Tamanhos de arquivo mínimos
  • ✓ Tempos de carregamento instantâneos
  • ✓ Hierarquia visual clara
  • ✓ Codificação de cores consistente

Modo Apresentação

Assets polidos e de alta fidelidade para demos, trailers e showcases. Texturas detalhadas, efeitos de partículas, música adaptativa e elementos de UI profissionais. Visuais com qualidade de produção prontos para uso.

  • ✓ Texturas & sprites em HD
  • ✓ Áudio adaptativo em várias camadas
  • ✓ Biblioteca de VFX de partículas
  • ✓ Componentes de UI temáticos

Famílias de Templates

Cada família de templates inclui pacotes de conteúdo Protótipo e Apresentação, cobrindo uma ampla variedade de gêneros de jogos.

Ação & Combate

Personagens, ambientes, efeitos e áudio para gameplay de ritmo acelerado

Estratégia & Lógica

Mapas, elementos de UI e assets para jogos de estratégia e quebra-cabeça

Aventura & Exploração

Mundos, tilesets e áudio atmosférico para experiências de mundo aberto

Personalizado

Assets base flexíveis para construir qualquer gênero de jogo

Áudio Integrado

Os pacotes de conteúdo incluem uma experiência de áudio completa, com música de fundo, efeitos sonoros, áudio de UI e elementos adaptativos que respondem ao gameplay em tempo real.

Acesse os Pacotes de Conteúdo

Pacotes de conteúdo, guias de integração de assets, especificações de manifesto e documentação completa estão incluídos no acesso ao SDK.
