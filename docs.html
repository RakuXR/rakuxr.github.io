AI Integration

AI platforms generate .raku files using our published schema. Point any LLM at rakuai.com/schema.json for the full .raku file specification, or rakuai.com/api-manifest.json for the curated API reference.

Supported AI Platforms

Full LLM Integration Guide → copy-paste prompts, troubleshooting, and auto-fix tools

ChatGPT (OpenAI)

GPT-4 and GPT-4o fully support .raku file generation. Always point it at rakuai.com/schema.json first to avoid hallucinated fields.

Claude (Anthropic)

Claude Opus and Sonnet models generate syntactically correct .raku files. Supports MCP server integration for enhanced tooling.

Gemini (Google)

Gemini Pro and Ultra support .raku schema. Best for iterative game design with multi-turn conversations.

GitHub Copilot

VS Code and Visual Studio extensions provide .raku file completion and validation. Open schema.json in an adjacent tab for autocomplete.

Example .raku File

This is what AI generates when you describe a game. All fields are documented in the schema.

{
"schema_version": "1.0",
"game": {
"title": "Space Defender",
"genre": "shooter",
"template": "space-shooter"
},
"ai": {
"dda_enabled": true,
"target_flow_state": 0.7,
"profiler_mode": "adaptive",
"emotional_tracking": true
},
"entities": [
{ "type": "player_ship", "health": 100 },
{ "type": "enemy_wave", "count": 10 }
],
"ai_disclosure": {
"live_generated": true,
"pre_generated": true,
"description": "AI adjusts difficulty based on player performance.",
"systems": ["dda", "adaptive_audio", "emotional_state"],
"guardrails": { "profanity_filter": true, "content_rating": "E", "report_endpoint": "https://api.raku.games/v1/reports" }
}
}

Asset paths: engine:// for built-in assets, user:// for player files, or absolute paths like C:/MyArt/hero.png. All engine:// assets ship with the runtime.

Diagnostics

RakuDiag is a built-in diagnostic tool for the Raku runtime — think DXDiag, but for Raku. It collects system information, runtime status, GPU capabilities, and AI subsystem health into a single report.

1

Launch RakuDiag

Open the Start Menu and navigate to RakuAI → Raku Diagnostic Tool. The tool scans your system and displays hardware, driver, and runtime details.

2

Review the Report

RakuDiag shows GPU info, DirectX feature level, OpenXR runtime status, installed modules with versions, AI subsystem readiness, and any detected issues or warnings.

3

Save All Information

Click "Save All Information" to export the full diagnostic report as a text file. Attach this file when filing bug reports to help the team reproduce and fix issues quickly.

Found a bug or need help? Post your diagnostic report on GitHub Discussions and the community or team will assist you.
