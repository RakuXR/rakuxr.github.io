Diagnostics

RakuDiag is a built-in diagnostic tool for the Raku runtime — think DXDiag, but for Raku. It collects system information, runtime status, GPU capabilities, and AI subsystem health into a single report.

1

Launch RakuDiag

Open the Start Menu and navigate to RakuAI → Raku Diagnostic Tool. The tool scans your system and displays hardware, driver, and runtime details.

2

Review the Report

RakuDiag shows GPU info, DirectX feature level, OpenXR runtime status, installed modules with versions, AI subsystem readiness, and any detected issues or warnings.

3

Save All Information

Click "Save All Information" to export the full diagnostic report as a text file. Attach this file when filing bug reports to help the team reproduce and fix issues quickly.

Found a bug or need help? Post your diagnostic report on GitHub Discussions and the community or team will assist you.
