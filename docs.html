Getting Started

1

Get the Schema

The .raku file schema defines the game format. Point any AI at it, or browse the sample games on GitHub to see the format in action.

2

Describe Your Game to AI

Open any AI assistant and say: "Using the Raku game engine schema at rakuai.com/schema.json, create a space shooter with adaptive difficulty." The AI generates a complete .raku file.

3

Validate & Play

Validate your .raku file with the validator tool, then double-click to play. Download the runtime →

AI Integration

AI platforms generate .raku files using our published schema. Point any LLM at rakuai.com/schema.json for the full .raku file specification, or rakuai.com/api-manifest.json for the curated API reference.

Supported AI Platforms

Full LLM Integration Guide → copy-paste prompts, troubleshooting, and auto-fix tools

ChatGPT (OpenAI)

GPT-4 and GPT-4o fully support .raku file generation. Always point it at rakuai.com/schema.json first to avoid hallucinated fields.

Claude (Anthropic)

Claude Opus and Sonnet models generate syntactically correct .raku files. Supports MCP server integration for enhanced tooling.

Gemini (Google)

Gemini Pro and Ultra support .raku schema. Best for iterative game design with multi-turn conversations.

GitHub Copilot

VS Code and Visual Studio extensions provide .raku file completion and validation. Open schema.json in an adjacent tab for autocomplete.

Example .raku File

This is what AI generates when you describe a game. All fields are documented in the schema.

{
"schema_version": "1.0",
"game": {
"title": "Space Defender",
"genre": "shooter",
"template": "space-shooter"
},
"ai": {
"dda_enabled": true,
"target_flow_state": 0.7,
"profiler_mode": "adaptive",
"emotional_tracking": true
},
"entities": [
{ "type": "player_ship", "health": 100 },
{ "type": "enemy_wave", "count": 10 }
],
"ai_disclosure": {
"live_generated": true,
"pre_generated": true,
"description": "AI adjusts difficulty based on player performance.",
"systems": ["dda", "adaptive-audio", "emotional-tracking"]
}
}
