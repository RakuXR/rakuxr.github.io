Connect a capture to your AI assistant

One-tap connect for Claude (Anthropic), ChatGPT (OpenAI), and Gemini (Google). No JSON config, no local Python. Backed by /api/v1/connect/* on raku-api.

Developers who want the local stdio MCP server can still find it in raku-runtime/mcp. That path is unchanged and remains the dev option.

1. Sign in

To attach an assistant to a capture we need to know which account it belongs to. Sign in with a Bearer JWT (from the website) or a raku_ API key (from /api/v1/onboard). Credentials live in this tab only and are never written to disk.
