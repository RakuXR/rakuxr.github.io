內容包

為每個範本精心策劃的素材 — 兩種模式，零猜測。

原型模式 vs 簡報模式

每個範本都附帶兩個內容包。選擇最符合您開發階段的那一個。

原型模式

專為快速迭代設計的輕量級佔位素材。簡單形狀、純色與基本音效，讓您不必等待美術就能測試遊戲邏輯。最適合 game jam 與早期原型開發。

  • ✓ 最小檔案大小
  • ✓ 即時載入
  • ✓ 清晰的視覺層次
  • ✓ 一致的色彩編碼

簡報模式

為示範、預告片與展示而生的精緻高保真素材。細緻貼圖、粒子特效、自適應音樂與專業 UI 元素。開箱即用的出貨等級視覺效果。

  • ✓ HD 貼圖與精靈
  • ✓ 多層自適應音訊
  • ✓ 粒子視覺特效庫
  • ✓ 主題化 UI 元件

範本系列

每個範本系列都同時包含原型與簡報內容包，涵蓋廣泛的遊戲類型。

動作與戰鬥

為節奏明快的遊玩提供角色、環境、特效與音訊

策略與邏輯

為策略與解謎類遊戲提供地圖、UI 元素與素材

冒險與探索

為開放式體驗提供世界、圖磚集與氛圍音訊

自訂

用於建構任何遊戲類型的彈性基礎素材

整合音訊

內容包提供完整的音訊體驗，包含背景音樂、音效、UI 音訊，以及能即時回應遊玩的自適應元素。

取得內容包

內容包、素材整合指南、清單規格與完整文件皆隨 SDK 存取權一併提供。
