1. 定義

「機密資訊」係指透過 RakuAI 開發者計畫所提供之任何非公開技術資訊、文件、API 規格、SDK 原始碼、遊戲設定結構（.raku 格式）及專有演算法。

2. 義務

  1. 你同意對所有機密資訊嚴格保密。
  2. 你不得對任何 RakuAI 軟體、SDK 或執行環境二進位檔進行還原工程、反編譯或反組譯。
  3. 你不得將 API 金鑰、SDK 版本或內部文件分享給未經授權的第三方。
  4. 你僅可將開發者工具用於開發與 RakuAI 平台整合之應用程式。

3. 允許用途

  • 使用 RakuAI API 建置應用程式與遊戲
  • 與你的開發團隊分享公開可取得的文件
  • 發行以 RakuAI 建立之遊戲（須附適當的 AI 揭露）
  • 向 RakuAI 團隊提供回饋與 bug 回報

4. 期間

本協議於你參與開發者計畫期間有效，並於終止後繼續有效兩（2）年。

5. 終止

任一方得以 30 日書面通知終止本協議。終止後，你必須銷毀手上所有機密資訊。

6. 準據法
