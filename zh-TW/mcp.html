運作方式

您的 AI 助理會說 MCP。RakuAI 就是這個連線另一端的執行階段， 向眼鏡、頭戴裝置與瀏覽器驅動真實的空間輸出。

六個工具

完整的 MCP 介面。結構驗證、稽核紀錄、權限控管。

對比 Unity MCP

Unity MCP（剛發布）：場景操作、資產擺放、 建置自動化 — Claude Desktop 在設計階段產生靜態場景。

RakuAI MCP（自 2026 年 3 月起進入生產）：世界模型編排 — 任何 LLM 在每一個影格將意圖串流進模擬步進。

Unity 是您的 AI 拿起來用的工具。RakuAI 則是您的 AI 棲身的身體。

為什麼 MCP 原生很重要

  1. 無需客製化整合 — 只要您的模型會說 MCP，它就會說 RakuAI。
  2. 設計上即多廠商 — Claude、ChatGPT、Gemini、Copilot 全都驅動同樣的六個工具。
  3. 生產就緒 — 預設拒絕、稽核紀錄、結構驗證。
