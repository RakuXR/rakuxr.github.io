搶先體驗 — 測試版

掃描空間或物體。
發布可互動的splat。
與全世界分享。

Raku Capture 將手機掃描轉化為 AI 助理可以對話的高斯splat — 然後為你提供可分享連結，任何人都能檢視、嵌入或將其放入自己的引擎。

免費開始擷取 瀏覽已發布的splat

測試期免費 • 手機或專業級相機 • 無需帳戶即可試用

三步從掃描到可分享的splat

擷取到資產的基礎功能已於測試版上線。第 1–3 步目前可用；第 3 步的 AI 編輯循環將於第 2 階段深化。

📱

1. 擷取

用手機環繞空間或物體移動。Raku Capture 記錄你的手機或專業級相機所能提供的影格 — 無需掃描設備，無需專業軟體。

⚙️

2. 重建

我們的流水線將你的影格轉換為高斯splat（.ply / .spz），並正規化為適合引擎的比例與座標系。基於 GPU 的重建；品質隨硬體可用性而提升。

🔗

3. 發布與分享

發布到你的創作者資料並取得可分享連結。任何人都能檢視 splat、嵌入或將其匯出到 Unity、Unreal 或 WebXR。你的擷取歸你所有 — 隨時可匯出。

MCP 控制層

與你擷取的場景對話

每個已發布的 splat 也是可由 MCP 驅動的場景。連接 Claude、GPT-4o 或任何相容 MCP 的模型，對你擷取的空間發出自然語言指令 — 17 個原生工具加上對約 9,500 個 API 端點的透傳存取。

深度場景編輯（替換物體、重新打光、放置/移除）是第 2 階段的交付內容，目前尚未完全發布 — 但 MCP 連接與基本工具呼叫現已可用。

藍圖 — 第 2 階段

Meshy 生成循環

Phase 2 wires in Meshy.ai 的照片級家具與房間重新設計生成。擷取一個空房間；發出 MCP 指令進行佈置或重新設計；Meshy 生成的資產由 AI 放入 splat 場景，並匯出為可直接用於引擎的格式。

合作方向真實且範圍明確。完全發布的端到端重新設計循環是第 2 階段的交付內容 — 而非當前功能。

藍圖 — 第 3 階段

創作者市集

第 3 階段將新增發布/消費市集：資產版本管理、授權中繼資料，以及每個已發布擷取的來源資訊。創作者分潤取決於真實的市集交易量與正式版定價。

目前沒有分潤系統。無論市集狀態如何，你的擷取始終可匯出，並始終歸你所有。

加入創作者搶先體驗

測試期間擷取免費。Splat 輸出為 .ply 與 .spz；可分享的資料連結；匯出始終開啟。搶先體驗成員將於第 2 階段 MCP 編輯與 Meshy 重新設計循環發布時優先取得存取權。
