API Documentation

The RakuAI REST API lets you generate games, iterate on designs, and manage game sessions programmatically. All endpoints require JWT authentication.

Available Endpoints

  • POST /api/generate Generate a new game from a text prompt
  • POST /api/iterate Iterate on an existing game with modifications
  • GET /api/games List and manage your generated games

Authentication

All API requests must include a valid JWT token in the Authorization header. Tokens are issued upon developer approval and can be managed through your developer dashboard.

Authorization: Bearer <your_jwt_token>
